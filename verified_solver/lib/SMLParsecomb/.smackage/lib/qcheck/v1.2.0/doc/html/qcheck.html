QCheck/SML

This manual describes QCheck/SML (version 1.1), an automatic testing library for Standard ML. It is modeled after the QuickCheck library for Haskell (ICFP 2000) by Koen Claessen and John Hughes, with many thanks. QCheck/SML (including this manual) is free software.
